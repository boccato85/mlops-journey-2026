# 🚀 MLOps Journey 2026

Este repositório é o registro da minha transição de carreira de **CloudOps** para **Engenharia de MLOps**. Aqui documento minha jornada técnica, certificações da CNCF e a implementação de pipelines de Machine Learning em larga escala.

## 🛠️ Stack Tecnológica & Hardware
- **OS:** Fedora Linux
- **Hardware:** Acer Predator (i7 14th Gen | 32GB RAM | RTX 4070 8GB)
- **Foco:** Kubernetes (K8s), Python, GitOps e Observabilidade.

## 🗺️ Roadmap de Evolução
Minha trajetória está dividida em 4 fases estratégicas:

1. **Fundação:** Python Avançado e Certificação KCNA.
2. **Infraestrutura:** Especialização em Kubernetes (Certificação CKA).
3. **Ciclo de Vida de ML:** MLOps Zoomcamp (MLflow, DVC, Prefect) e Certificação PCA (Prometheus).
4. **Especialização:** GitOps com ArgoCD (CAPA), Kubeflow e CKAD.
